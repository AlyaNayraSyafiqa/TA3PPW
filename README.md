# KweenyTech – Landing Page
Landing page responsif untuk perusahaan teknologi bernama **KweenyTech**, dibuat menggunakan **Tailwind CSS**.

## Deskripsi
Proyek ini merupakan landing page modern yang dirancang untuk menampilkan profil perusahaan teknologi.  
Halaman ini terdiri dari beberapa bagian penting seperti header, hero, fitur utama, tentang perusahaan, form kontak, dan footer.

## Teknologi yang Digunakan
- **HTML5**
- **Tailwind CSS (via CDN)**
